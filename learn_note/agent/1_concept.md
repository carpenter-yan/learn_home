

1. 什么是智能体  
在人工智能领域，智能体被定义为任何能够通过传感器（Sensors）感知其所处***环境***（Environment），并***自主***地通过执行器（Actuators）采取***行动***（Action）以达成特定目标的实体。

2. 五种传统智能体  
2.1. 反射智能体（Simple Reflex Agent）:基于预设的"条件-动作"规则，仅依赖当前感知输入直接做出反应，无记忆、无预测，如自动恒温器。  
2.2. 基于模型的反射智能体（Model-Based Reflex Agent）:在反射智能体基础上引入内部世界模型，能追踪和推断环境中无法直接感知的状态，使决策基于持续的世界理解而非瞬时感知，如隧道中的自动驾驶汽车。  
2.3. 基于目标的智能体（Goal-Based Agent）:在内部模型的基础上引入明确目标，通过搜索与规划，主动选择能导向目标状态的行动路径，如GPS导航系统。  
2.4. 基于效用的智能体（Utility-Based Agent）:为每个世界状态赋予效用值（满意度），在多个冲突目标之间进行权衡，以最大化期望效用作为决策准则，如多功能导航系统。  
2.5. 学习型智能体（Learning Agent）:通过与环境的互动和反馈自主修正决策策略，无需依赖人类预设的规则或模型，具备从经验中自我进化的能力。如AlphaGo Zero。

3. LLM智能体工作方式
- 规划与推理：智能体首先会将这个高层级目标分解为一系列逻辑子任务，这是一个内在的、由模型驱动的规划过程。
- 工具使用：在执行规划时，智能体识别到信息缺口，会主动调用外部工具来补全。
- 动态修正：在交互过程中，智能体会将用户的反馈视为新的约束，并据此调整后续的行动。

4. LLM智能体与传统智能体的本质区别  
传统智能体的能力源于工程师的显式编程与知识构建，其行为模式是确定且有边界的；而LLM智能体则通过在海量数据上的预训练，获得了隐式的世界模型与强大的涌现能力，使其能够以更灵活、更通用的方式应对复杂任务。

5. 智能体分类  
5.1. 基于内部决策架构的分类 见2  
5.2. 基于时间与反应性的分类  
    - 反应式智能体 (Reactive Agents):2-1,2-2  
    - 规划式智能体(Deliberative Agents):2-3,2-4  
    - 混合式智能体(Hybrid Agents) :2-5  
5.3. 基于知识表示的分类:智能体用以决策的知识，究竟是以何种形式存于其“思想”之中  
    - 符号主义 （Symbolic AI）:其核心信念是智能源于对符号的逻辑操作。
    - 连接主义 （Sub-symbolic AI）:知识是内隐地分布在一个由大量神经元组成的复杂网络中。神经网络和深度学习是其代表。
    - 神经符号主义（Neuro-Symbolic AI）:融合两大范式的优点，创造出一个既能像神经网络从数据中学习，又能像符号系统进行逻辑推理的混合智能体。

6. 智能体的运行机制:感知-思考-行动-观察(输入+思考+输出)
    - 感知 (Perception)/观察 (Observation):可以是用户的初始指令，也可以是上一步行动所导致的环境状态变化反馈。
    - 思考 (Thought):规划 (Planning)+工具选择 (Tool Selection)
    - 行动 (Action):决策后，通过其执行器（Actuators）行动。表现为调用工具，从而改变环境状态。

7. 智能体应用的协作模式  
   7.1 作为开发者工具（AI 编程辅助工具）：一种是作为高效工具，深度融入我们的工作流
   - GitHubCopilot
   - Claude Code
   - Trae
   - Cursor  
   7.2 作为自主协作者:智能体会像一个真正的项目成员一样，独立地进行规划、推理、执行和反思，直到最终交付成果。
   - 单智能体自主循环:其核心是一个通用智能体通过“思考-规划-执行-反思”的闭环，不断进行自我提示和迭代，以完成一个开放式的高层级目标。
   - 多智能体协作:这是当前最主流的探索方向，旨在通过模拟人类团队的协作模式来解决复杂问题。
   - 高级控制流架构:诸如 LangGraph 等框架，则更侧重于为智能体提供更强大的底层工程基础。
